# Weather

Виджет погоды.
Версия: v. 0.0.1

### Технологии: 
- JS 
- React
- Redux
- GIT
- Node.js
- Webpack
- HTML
- CSS
- BEM

### Установка:

Клонировать [репозиторий](https://github.com/InInferno/weather-react-redux.git)

Установить пакеты

    npm install

### Запуск:

Сборка production

      npm run build
      
Режим разработки с hotreload

    npm run start
